# ToadBot Guide (Static)

Готовый статический сайт для GitHub Pages: короткий, удобный гайд по ToadBot.

- Все факты/цифры/команды взяты из официального гайда: https://toadbot.info/guide
- Все кнопки «Начать» ведут на: https://t.me/toadbot?start=228691787

## Как развернуть на GitHub Pages

1) Загрузите содержимое репозитория (файлы `*.html` и папку `assets/`).
2) В GitHub откройте **Settings → Pages**.
3) Source: **Deploy from a branch**
4) Branch: **main** / folder: **/(root)**
5) Сохраните — сайт будет доступен по вашему GitHub Pages URL.

## Структура

- `index.html` — главная
- `start.html` — быстрый старт
- `modes*.html` — режимы (арена/тусы/подземелья/ограбления/соревнования/ежедневки)
- `progress.html` — прокачка
- `clans.html` — кланы
- `commands.html` — список команд с поиском и копированием
- `admin.html` — настройки для админов чатов
- `faq.html` — нюансы и ответы
- `assets/` — стили/скрипт/иконки

## Примечание

Если официальный гайд обновится — обновляйте тексты на страницах.
