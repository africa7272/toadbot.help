Начать за 2 минуты

Короткий путь из «бот добавлен» → «у тебя есть жаба». Команды из официального гайда.

1) Добавь бота в чат и выдай админку

Сначала добавь @toadbot в беседу, затем выдай ему права администратора — и только потом переходи к игровым действиям.

2) Подпишись на рассылку жаб

Подписка нужна, чтобы в чат приходили жабки. Статус подписки можно проверить командой Подписаться на рассылку.

Если только подписались — задержка может быть до 5 часов. Потом рассылка идёт раз в два часа по расписанию.

3) Забери жабу

Когда бот пришлёт жабу, нужно быть первым и забрать её командой Взять жабу.

  • Для новых чатов в течение 24 часов жабу может брать каждый желающий (ограничение: одна жаба в сутки).
  • Спустя 24 часа можно взять только ту жабку, которую прислал бот. Если её уже взяли — ждём следующую.

4) Если играть не с кем

Можно зайти в официальный игровой чат: он находится через команду Помощь (внизу будет ссылка).
